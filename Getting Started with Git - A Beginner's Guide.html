Git Made Simple: A Beginner's Guide

What is Git?
Git is a version control system that helps you track changes in files and collaborate with others. It is widely used in software development but is also great for any document versioning.

What is GitHub?
GitHub is a website that lets you store your Git repositories online and work with others easily.

Basic Git Commands

Check current folder:
pwd
Shows where you are inside your computer.
List files:
ls
Lists all the files in the current folder.
Move to another folder:
cd folder-name
Navigates to a specific folder.
Create a new folder:
mkdir my-folder
Creates a new directory (folder).
Initialize Git:
git init
Starts version control in your folder.
Check Git status:
git status
Shows which files are changed, added, or ready to be saved.
Add all changes:
git add .
Prepares your changes to be saved.
Save your changes:
git commit -m "Your message"
Commits your changes with a message (like a save note).
Push to GitHub:
git push
Uploads your saved work to GitHub.
Check change history:
git log
See who changed what and when.
Download changes from GitHub:
git pull
Gets updates from the online repository.

Tips

  • Use arrows ↑↓ to scroll through past commands.
  • Use tab to autocomplete file or folder names.
  • Always write a meaningful message when you commit.
  • If you make a mistake, git restore can help undo it.
